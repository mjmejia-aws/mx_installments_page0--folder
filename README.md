#MX Installments - Wekly GMS Dashboard 

Streamlit dashboard to analize Weekly Installments GMS with Year-over-Year (YoY) comparation

# What this app does
- Displays weekly GMS treds
- Shows YoY GMS comparason for the last 5 weeks
- Interactive filters (date range, program, ASP band, channel, GL product)
- Allows CSV export of aggregated data
- Allows HTML export of charts

1. Install dependencies:

2. Run the app:
